# chrome-extension template

Vite + Vue 3 + TypeScript + Tailwind 4 + Element-Plus chrome extension initialization template

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
npm run test:unit
```
